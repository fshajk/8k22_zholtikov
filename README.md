## Описание
Веб‑приложение для онлайн‑продажи строительных материалов. Реализованы:
- каталог товаров с фильтрацией;
- корзина покупок;
- оформление заказа;
- аутентификация пользователей (вход/регистрация);
- админка для управления товарами и заказами.

## Функциональные возможности
1. **Пользователи:**
   - регистрация и авторизация;
   - просмотр списка заказов;
   - детализация заказа.
2. **Магазин:**
   - каталог товаров с пагинацией;
   - страница товара с описанием;
   - добавление в корзину;
   - редактирование корзины;
   - оформление заказа (форма с контактами и адресом).
3. **Администрирование:**
   - управление товарами (добавление, редактирование, удаление);
   - просмотр и управление заказами.
